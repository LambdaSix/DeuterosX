What do you need?

The only essential thing you really need to do Deuteros Reborn modding is a text editor - windows notepad will do just fine. Of course, a handy thing to have is the game, so that you can debug the mod and as well use the integrated runtime game editor (we'll look on that later). And of course, if you are making a custom mod, you probably want to use your own graphics, sounds and whatever else, so be ready to make any of this stuff you'd love to include in your mod.

The next thing you can use is the Deuteros Reborn Engine's (DRE's) online documentation, though you may have problems with using it the first time - unless you are used to MSDN-style documentation and you have a basic understanding of Microsoft.NET - the documentation is designed for advanced modders (as explained in the Advanced Handbook)

Deuteros Reborn Engine Architecture

DRE's modding is done in two separate levels - the first one is editting the Xml files located in directory Mods\<modname>. The other one is a little more complicated (though not necessary), requiring you to use a Microsoft.NET language - explained in the Advanced Handbook.

In-game font definition file (Fonts.fdrf)

<FontList>
<Font>
<Id>fntMain</Id> - the id of this font
<Source>SystemFont</Source> - the source. May be SystemFont or BitmapFont
<Params>
<FontName>Microsoft Sans Serif</FontName> - only for SystemFont. Specifies the system font name
<FontSize>8.25</FontSize> - only for SystemFont. Specifies the size of the font
<FontStyle>Regular</FontStyle> - only for SystemFont. Specifies the font style
</Params>
</Font>
<FontList>

Mod information file (GameInfo.fdri)

<GameInfo>
<Title>Deuteros Reborn</Title>
<ShowEngineVersion>True</ShowEngineVersion>

</GameInfo>
